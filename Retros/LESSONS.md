## Design

### 디자인 가이드 문서와 구현 코드를 별도 대조 검증하라 #coding #design
디자인 가이드(`shooter-design-guide.md`)에 "Green은 UI 상태 전용"이라 명시되어 있었지만 실제 CSS에는 네비 언더라인에 `#00d255`가 사용되어 있었다. 가이드 문서 작성 후 구현 단계에서 위반이 발생하므로, 코드 리뷰 시 가이드 원칙 항목과 CSS 변수 사용처를 대조 확인해야 한다.

## Copywriting

### 자기선언 순위보다 검증 가능한 숫자가 신뢰를 만든다 #strategy #copywriting
"국내 1위"처럼 근거 없는 순위 주장은 오히려 신뢰를 낮춘다. "2,300+ 브랜드 런칭", "재의뢰율 73%" 같이 구체적이고 검증 가능한 수치로 대체하면 방문자가 스스로 판단할 수 있어 전환율에 더 효과적이다.
